# Installation

---

## Requirements

- macOS 13+ (Ventura or later)

---

## Install

1. Download the DMG from [GitHub Releases](https://github.com/qtld88/FilmCan/releases)
2. Drag `FilmCan.app` to **Applications**
3. Open FilmCan (right-click > Open on first launch)

---

## Permissions

Grant these in System Settings:

- **Notifications**: get alerts when transfers finish
- **Full Disk Access**: read cards and write to drives

Paths:
- **System Settings** > **Privacy & Security** > **Full Disk Access**
- **System Settings** > **Notifications** > FilmCan

---

## Update

Download the latest DMG and drag it to **Applications** to replace.

---

## Uninstall

1. Move FilmCan.app to Trash
2. Optional, remove settings:
   ```bash
   rm -rf ~/Library/Application\ Support/FilmCan
   ```

---

## Need Help?

See [Troubleshooting](./troubleshooting.md) if something goes wrong.
