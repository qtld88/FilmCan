# AI Usage Disclosure

FilmCan is **not** an AI product and does not call AI APIs at runtime.

During development, AI tools were used to assist with:
- Writing code
- Drafting documentation
- Refactoring and code cleanup suggestions
- Reviewing error messages and UX copy

All changes are reviewed and curated by the maintainers before release.

No user data is ever sent to AI services by the app.
