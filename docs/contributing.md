## Code Contributions

**Requirements**: macOS 13+, Xcode 15 (Swift 5.9)

```bash
git clone https://github.com/qtld88/FilmCan.git
cd FilmCan
open FilmCan/FilmCan.xcodeproj
```

**Test checklist:**
- Small and large files
- Multiple destinations
- Stop/resume
- Verification + hash lists
- Organization presets
