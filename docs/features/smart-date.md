# Smart Date

Control which date is used in folder/file templates.

---

## When to Use

- Shooting days cross midnight
- Camera clocks are wrong
- You need consistent dates across multiple cards

---

## Set the Shoot Day

In **Destinations**, set **Custom date for tokens** to the desired day and time.

Example:
- Shooting starts March 14 at 8pm
- You wrap March 15 at 2am
- Set Smart Date to "March 14"
- All files use `{date}` = `20260314`
